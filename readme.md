#Suit-baze
Suit-baze is a Front-end starter kit base on [HTML5 Boilerplate](https://github.com/h5bp/html5-boilerplate) modified by developers from [Suitmedia](https://github.com/Suitmedia).

## What's included

* Normalize
* Baze Grid
* Modernizr
* jQuery
* Fastclick
* Gulp build system
* Bower manifest

### Available gulp task

* Default
* watch
* build
* livereload
